2019年4月22日工作总结

本周进展:

一、利用周末时间看了CVPR 2019分享会的几个报告

1."AE^2：autoencoder in autoencoder"这个文章是在组会分享过的，又重新听了一遍报告，文章主要研究的是多图学习。主要动机还是针对多模态数据的特征提取，保证视图之间的相关性和独立性。外部的自动编码器用作提取不同模态数据之间的相关性，内部的自动编码器用作提取一个模态之间的独立性。这种自动编码器的相互约束可以避免直接对高维数据进行重建，融合各视图的本质信息。

2."Label-efficient semi-supervised learning via graph filtering"这篇文章介绍了一种关于图网络的新思路，主要可以避免图网络巨大的参数量。文章中最多使用两层网络，但是一层图卷积网络中可以有k个低通滤波器（效果等同于k层的图卷积网络）。k的值随着可用的标签样本的数量调节，可用标签样本越少，k越小。代码：https://github.com/liqimai/Efficient-SSL

3."Self-supervised convolutional subspace clustering network"

这篇文章是研究子空间聚类的，子空间聚类就是通过多个平面将一堆数据点划分到多个子空间中，谱聚类就是子空间聚类中一种常用的方法。“ Deep Subspace Clustering Network, NIPS 2017” 介绍了结合神经网络的子空间聚类方法。这篇文章就是根据其改进的一种新型结构，将谱聚类的结果反馈给神经网络进行参数调整。

4."Multi-label image recognition with graph convolutional network"

我看高老师也关注了这个文章，这篇文章首先需要将自然语言的标签形成GCN能识别的图结构，通过GCN的学习能够得到每一个类的分类器。然后输入图像经过传统CNN进行特征提取，特征输入到训练好的分类器中就会得到多标签的分类结果。思路比较简单。代码：https://github.com/chenzhaomin123/ML_GCN

5.“Learning to learn image classifiers with visual analogy”

这篇文章的原理比较简单，首先将训练数据经过Alex Net进行特征提取，然后通过计算特征的相似度构建图结构，然后通过Graph Embedding 将图中的每一个node映射为一个embedding特征。测试阶段对于一个全新的样本通过原有的图（图是可以生长的）估算出 其embedding特征然后进行分类。

6.“SeerNet：Predicting convolutional neural network feature-map sparsity through low-bit quantization”

这是一篇关于网络结构稀疏的报告作为科普会去看一下。
